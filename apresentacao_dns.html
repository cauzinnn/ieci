A INTERNET NÃO ESTÁ NA NUVEM!

  • Uma Rede Física: A internet é, na verdade, uma gigantesca teia mundial de computadores ligados por cabos físicos.
  • Cabos Submarinos: Grossos cabos de fibra óptica cruzam os oceanos bem no fundo do mar, ligando continentes em velocidade da luz.
  • Computadores e Clientes: O seu celular (cliente) envia sinais elétricos pelos cabos para pedir informações a outros computadores maiores (servidores).

02 // O Endereço IP
IP_ADDRESS_EXPLAINED

O QUE É O NÚMERO IP?

  • O Endereço da Máquina: Cada computador ou celular conectado na internet possui um endereço exclusivo chamado IP.
  • Formato Numérico: É um conjunto de números de 32 bits (4 bytes) como 186.230.134.12.
  • Como as Cartas no Correio: Para enviar dados de um aparelho para outro, você precisa saber o "endereço de destino" exato daquela máquina.

03 // O Método Antigo
THE_MANUAL_HOSTS_FILE

O ARQUIVO HOSTS.TXT

  • Mapeamento Manual: No início, toda a internet era mapeada em um único arquivo de texto estático, onde cada linha continha um IP e o nome correspondente.
  • Download via FTP: Para acessar outros computadores, sua máquina precisava baixar a versão mais recente desse arquivo de um servidor central diariamente.
  • O Colapso: Conforme a rede cresceu de dezenas para milhões (e depois bilhões) de dispositivos, ficou impossível atualizar e distribuir esse arquivo manualmente. A centralização virou o maior gargalo da rede!

04 // O DNS
DNS_THE_CONTACTS_LIST

A AGENDA DA INTERNET

  • Problema Humano: Ninguém decora números IP! Seria péssimo ter que lembrar "142.250.191.46" para abrir o Google.
  • Agenda de Contatos: No seu celular, você clica em "Mãe" em vez de digitar o número de telefone dela toda vez.
  • Sistema DNS: O DNS é essa agenda global. Ele recebe o nome do site que você digitou (ufrj.br) e traduz para o IP correspondente.

05 // A Busca Rápida
DNS_LOOKUP_JOURNEY

COMO A TRADUÇÃO É FEITA?

  • Você digita o nome: Digita "ufrj.br" e dá Enter.
  • Computador pergunta: Seu PC pergunta a um servidor centralizador local: "Ei, qual o número IP do site ufrj.br?".
  • Resposta viaja: O servidor local busca o número em uma rede de servidores maiores, acha o IP, devolve para o seu PC, e a foto carrega! Tudo isso em menos de 1 segundo!

06 // Tipos de Registro
DNS_RECORD_TYPES

TIPOS DE REGISTRO PRINCIPAIS

  • Registros A / AAAA: Converte o nome diretamente em um endereço IPv4 / IPv6 para as máquinas conversarem.
  • CNAME: Funciona como um apelido ("alias"). Se "www.ufrj.br" for acessado, ele redireciona para "ufrj.br".
  • MX (Mail Exchanger): Indica o servidor que recebe e-mails para aquele site específico.
